About This Project

This project explores a novel approach to diabetes risk assessment by integrating dermatoglyphics (fingerprint patterns) with traditional health metrics. We believe that unique patterns in fingerprints, combined with factors like age, BMI, and family history, can provide a more comprehensive and non-invasive preliminary screening for diabetes risk.

Concept

The core idea is based on the premise that certain physiological conditions, including metabolic disorders like diabetes, might manifest subtle correlations with dermatoglyphic features. While not a definitive diagnostic tool, this application aims to serve as an early warning system or a screening tool that prompts further medical consultation.

Workflow

  1. User Authentication: Users must first register and log in to access the prediction features, ensuring a personalized and secure experience.
  2. Fingerprint Image Upload: The user uploads a clear image of their fingerprint.
  3. Feature Extraction: Our backend (using OpenCV and custom algorithms) processes the uploaded image to automatically extract two key fingerprint features:
    • **Fingerprint Type:** Categorized as Arch, Loop, or Whorl.
    • **Ridge Count:** An estimation of the number of ridges in a specific area.
  4. Additional Health Data Input: The user then provides other relevant health information, including their Age, Body Mass Index (BMI), and Family History of Diabetes.
  5. Diabetes Risk Prediction: A pre-trained machine learning model (trained on a synthetic dataset combining fingerprint data with health metrics) processes all the collected inputs.
  6. Result Display: The application outputs a prediction of "Low," "Medium," or "High" diabetes risk based on the model's assessment.

Technology Stack

  • Backend: Flask (Python Web Framework)
  • Machine Learning: Scikit-learn (RandomForestClassifier)
  • Image Processing: OpenCV and Pillow
  • Database: SQLite (for user management)
  • Frontend: HTML, CSS (Tailwind CSS for styling), Jinja2 (Templating Engine)

*Disclaimer: This project is for educational and experimental purposes only and should not be used for actual medical diagnosis. Always consult with a healthcare professional for any health concerns.*
